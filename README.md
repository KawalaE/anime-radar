# Anime Radar	

Anime Radar is an anime discovery app that lets users explore a wide range of new anime, offering detailed information about each one. This app is built with `React`, using `Chakra UI` for the user interface. State management is handled globally through `Zustand`. 
`React Query` is used to manage data caching, loading states, error handling, infinite queries, and API interactions. A significant aspect of this project is the integration tests, which were implemented using `Vitest`. 
Mocking of the API requests for testing puroposes was accomplished with `Mock Service Worker`.

[Click here!](https://anime-radar.vercel.app/)
